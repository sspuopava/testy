A: Jazyk HTML a jeho verze

Jazyk HTML a jeho verze

Dopište chybějící slova nebo znaky. Dodržujte český (anglický) pravopis.

HTML (Hypertext Markup Language = hypertextový značkovací jazyk) je speciální jazyk pro publikování hypertextových dokumentů označovaných jako webové stránky.

Za hypertext považujeme text, který obsahuje propojení (odkazy) na jiné texty. Označení značkovací jazyk souvisí se strukturováním dokumentu pomocí značek (angl. tags).

První verzi jazyka HTML publikoval zakladatel služby WWW Tim Berners Lee v roce 1991. Tato verze umožňovala vkládat do textu obrázky, hypertextové odkazy, vytvořit několik logických úrovní a několik druhů zvýraznění. Byla označena jako HTML 0.9. Požadavky uživatelů se postupně zvyšovaly a promítaly se do vývoje následujících verzí HTML.

Nástupcem velmi rozšířené verze HTML 4.01, která vycházela z metajazyka SGML, se stal jazyk XHTML, jehož základem je modernější metajazyk XML (eXtensible ML, rozšiřitelný značkovací jazyk).

Aktuální verze jazyka HTML se označuje jako HTML 5. Definitivně ukončila závislost na SGML, opravila mnoho chyb předešlých verzí, vyřadila řadu zastaralých a nepoužívaných prvků a zavedla nové sémantické prvky. Zároveň přibyla podpora moderních technologií (multimédia), byl zaveden nový systém vývoje jazyka.

O výběru definice HTML rozhoduje speciální element DOCTYPE, který se uvádí na začátku kódu stránky a v případě XHTML obsahuje odkaz na příslušný DTD (Document Type Definition). DTD představuje návod, jak má dokument, napsaný ve značkovacím jazyce, interpretovat prohlížeč (tj. např. jaké elementy dokument používá, jak s nimi zacházet, jaký význam mají jednotlivé atributy apod.).

Definiční dokument je umístěn na serveru konsorcia W3C (World Wide Web Consorcium), které má na starost definování standardů služby WWW.

B: Syntax HTML

Syntax HTML

Vyberte ze seznamu vhodný pojem

Strukturální značky definují strukturu dokumentu (např. <p> <h1>, <hr>).

Sémantické značky popisují povahu obsahu prvku (<header>, <address>>, <article>), usnadňují automatizované zpracovávání dokumentů a vyhledávání informací na webu.

Stylistické značky určují vzhled prvku při zobrazení (např. <b>, <i>), postupně se od nich upouští, trendem je používání kaskádových stylů, které vzhled popisují odděleně od obsahu dokumentu.

Blokové elementy po takovém prvku je text dokumentu zalomen (odřádkován); řadíme k nim například nadpisy, odstavce, oddíly, seznamy nebo tabulky.

Inline elementy nacházejí se uvnitř textu a nedochází po nich k zalomení, obvykle plní funkci zvýraznění nějaké části textu.

Nahrazované elementy jsou zaměněny nějakým obsahem, pro začlenění dokumentu jsou důležité jejich rozměry (např. obrázky nebo multimediální soubory).

Atributy přiřazují nějaké vlastnosti určitému elementu, píší se do počátečního tagu mezi uvozovky, může jich být i více (oddělují se mezerou).

Entity vyjadřují pomocí sekvence znaků jeden určitý znak nebo symbol, který je obvykle z určitého důvodu netypický (nelze ho jednoduše zapsat pomocí klávesnice) nebo problematický, protože se využívá k jiným účelům v jazyce HTML.

C : Struktura webové stránky

Struktura webové stránky

Přesuňte jednotlivé elementy do správného bloku HTML stránky.

Hlavička stránky <head>

  • <title>Název stránky</title>
  • <meta charset="utf-8">
  • <meta name="viewport" content="width=device-width, initial-scale=1">
  • <link rel="stylesheet" href="css/main.css"></link>
  • <style>* {position: relative}</style>

Tělo stránky <b>

  • <h1>Název stránky</h1>
  • <hr>
  • <footer>&copy; 2019</footer>
  • <div>Lorem, ipsum dolor sit.</div>
  • <br>
  • <img src="foto.jpg" alt="Fotka">

F : Validita kódu

Validita kódu

Zatrhněte ty řádky, které obsahují validní HTML5 kód.

  1. <img src="../obr/logo.png" alt="Firemní logo">
  2. <img src="logo.png" title="Firemní logo">
  3. <a src="index.html">Úvodní stránka</a>
  4. <a href="index.html" target="_blank">Úvodní stránka</a>
  5. <em><a href="index.html">Úvodní stránka</em></a>
  6. <a href="#galerie" style="color:red">Galerie</a>
  7. <em><a href="http://192.168.1.5/index.html">Úvodní stránka</a></em>
  8. <p>Text<br>odstavce</p>
  9. <p><center>Text odstavce</center></p>
